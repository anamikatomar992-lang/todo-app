# Smart To-Do App

## Description
This is a modern To-Do web application that helps users manage daily tasks efficiently. It supports adding, editing, deleting, and marking tasks as completed, with data stored using localStorage.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Persistent storage using localStorage
- Task timestamps
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in your browser
